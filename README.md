# shipment-delays-analysis

## Objective
Analyze shipment data to identify patterns and locations associated with delivery delays.

## SQL Analysis

### Shipment Status Summary
This query analyzes shipment data to determine how many shipments were delivered on time versus delayed.  
It groups shipments by delivery status and counts the total number in each category to provide a quick operational overview.

## Background
This project simulates a logistics environment where shipment delays impact operations and efficiency.
